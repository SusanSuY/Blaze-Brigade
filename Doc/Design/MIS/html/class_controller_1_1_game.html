override void Controller.Game.LoadContent ( )
inlineprotected

This method Loads all GUI and Map Textures, fonts, and SoundEffect Wav files. An instance is created for the wave files that are a song, then isLooped is set to true. Volume for the songs are also adjusted to half the original value. The method will then also call initializeGame() to set up the rest of the game.

§ Update()

override void Controller.Game.Update ( GameTime  gameTime)
inlineprotected

Updates game in real time at 60 times per second.

Update Components:

  • Checks if player clicks exit on the exit window
  • Calls MouseHandler to update mouse position if game is running
  • A switch case is used to update the game depending on what current Game State is.
  • During Game State Main Menu:
    1. Checks if start game button is clicked. In which case the main menu is closed, Menu Song is stopped, Map song is played, main game is set to visible, and Game State is set to Playing.
    2. Checks if get instruction button is clicked. In which case the gameState is switched to HowToPlay and instruction screen pops up.
    3. Checks if Exit Game is clicked, in which case the game closes.
  • During Game State HowToPlay:
    1. Checks if next button is clicked. In such case, change game state to HowToPlay2, set the 2nd instruction screen to show, and close the current window.
    2. Checks if back button is clicked. In such case, change game state to MainMenu, set the Main Menu to show, and close the current window.
    3. Checks if Exit Game is clicked, in which case the game closes.
  • During Game State HowToPlay2:
    1. Checks if next button is clicked. In such case, change game state to HowToPlay3, set the 2nd instruction screen to show, and close the current window.
    2. Checks if back button is clicked. In such case, change game state to HowToPlay1, set the 2nd instruction screen to show, and close the current window.
    3. Checks if Exit Game is clicked, in which case the game closes.
  • During Game State HowToPlay3:
    1. Checks if back button is clicked. In such case, change game state to HowToPlay2, set the 2nd instruction screen to show, and close the current window.
    2. Checks if Exit Game is clicked, in which case the game closes.
  • During Game State Playing:
    1. Loads the map texture
    2. If GameFunction isTurnOver is true, switch the current player and other player, set turn transition to true, and reset the camera to the new current player's turn.
    3. If transition turn is true, set it to false after 1.5 seconds.
    4. If currentPlayerDamagePopUp is true, set it to false after 2.5 seconds.
    5. If enemyPlayerDamagePopUp is true, set it to false after 2.5 seconds.
    6. Loops over all player1 and 2's units, and removes the ones that are dead. If either player has no units left, isGameOver is set to true, and Song changes from map theme to Game over theme.
      Parameters
      gameTimeThe current Game Time.
